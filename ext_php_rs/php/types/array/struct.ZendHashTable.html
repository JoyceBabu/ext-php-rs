Creates a new hash table wrapper. This will not be freed when it goes out of scope in Rust.

Parameters

  • ptr - The pointer of the actual hash table.
  • free - Whether the pointer should be freed when the resulting ZendHashTable goes out of scope.

Safety

As a raw pointer is given this function is unsafe, you must ensure that the pointer is valid when calling the function. A simple null check is done but this is not sufficient in most cases.
